# Vector Basis Finder using SymPy with a JavaSwing GUI

A desktop application that calculates whether a set of vectors is linearly independent and, if not, removes redundant vectors to determine a basis for the subspace they span. Built with **Java Swing** and **Python (SymPy)**.
